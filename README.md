# IMDB-api
for streamlit hackathon
uses the omdb api to get movie data
https://imdb-api.streamlit.app/

## How to run
1. Clone the repo
2. Install the requirements
3. Run the imdb_app.py file
    ```bash
    streamlit run imdb_app.py
    ```
4. Enjoy!
